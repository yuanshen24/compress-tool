# 压缩排除工具

一个带 Web UI 的压缩排除工具，可浏览目录、选择要排除的项，然后执行压缩。

## 功能

- **图形界面**：浏览器中运行，现代简洁的 UI，高 DPI 屏幕文字清晰
- **排除勾选**：扫描目录后默认排除全部，取消勾选即包含在压缩包中
- **子目录导航**：点击文件夹图标进入子目录，面包屑导航随时返回上级。目录行显示「已排除 N/M」标记，在上级目录也能看到子目录内的排除状态
- **排序**：支持按文件名或文件大小排序（点击列表表头切换升降序）
- **多格式支持**：tar.gz / tar.bz2 / tar.xz / zip
- **零外部依赖**：仅使用 Python 标准库，无需 pip install

## 运行

```bash
python3 compress-tool.py
```

浏览器会自动打开 `http://127.0.0.1:8765`，Ctrl+C 退出。

## 使用方式

1. 输入要压缩的根目录路径，点击「刷新」
2. 文件列表默认勾选全部（= 排除），取消勾选则包含该项
3. 点击文件夹图标（📁）可进入子目录继续勾选。面包屑导航或「← 返回上级」可回到上层目录
4. 点击「大小」或「文件名」表头可按对应列排序
5. 选择压缩格式，输入输出文件名
6. 点击「开始压缩」

输出文件默认放在根目录的上一级目录中。

## 系统要求

- Python 3.6+
- tar / gzip / zip（Linux 系统自带）

## 截图说明

界面从上到下分为五个区域：

| 区域 | 说明 |
|------|------|
| 根目录选择 | 输入路径 + 刷新按钮 |
| 导航栏 | 返回上级按钮 + 面包屑路径 + 排除计数 |
| 工具栏 | 全选/全不选/反选 + 压缩格式选择 |
| 文件列表 | 表头可点击排序，文件夹图标可点击进入子目录 |
| 操作栏 | 输出文件名 + 开始压缩按钮 + 状态指示 |
